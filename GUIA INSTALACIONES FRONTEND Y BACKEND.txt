Backend
 - JDK 17 y configurar variable de entorno JAVA_HOME
   - Descarga: https://download.oracle.com/java/17/archive/jdk-17.0.4_windows-x64_bin.exe
   - Instalación: https://youtu.be/0FXl5rBwNVk   se explica con Java 18 pero el procedimiento es similar para java 17

 - IntelliJ IDEA Ultimate
   - Descarga: https://www.jetbrains.com/es-es/idea/download/#section=windows
